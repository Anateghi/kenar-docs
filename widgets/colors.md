# رنگ‌های دیوار

در اینجا رنگ‌های قابل استفاده در ویجت‌های دیوار قابل مشاهده هستند.

```text
SUCCESS_PRIMARY
SUCCESS_SECONDARY
WARNING_PRIMARY
WARNING_SECONDARY
ERROR_PRIMARY
TEXT_PRIMARY
TEXT_SECONDARY
TEXT_HINT
TEXT_DIVIDER
ICON_PRIMARY
ICON_SECONDARY
ICON_HINT
ICON_DIVIDER
WHITE_PRIMARY
```

از این رنگ‌ها میتوان برای تنظیم رنگ متون و آیکان‌ها استفاده نمود.
